### 1. WorkFlow - 系统在线状态控制

#### 添加的属性和事件
```csharp
/// <summary>
/// 系统在线状态
/// </summary>
public bool IsOnline { get; private set; } = false;

/// <summary>
/// 系统在线状态变化事件
/// </summary>
public event Action<bool> OnlineStateChanged;
```

#### 核心方法
```csharp
/// <summary>
/// 设置系统在线状态
/// </summary>
public void SetOnlineState(bool online)
{
  if (IsOnline == online) return;
    
    IsOnline = online;
    LogHelper.Info($"系统状态切换为: {(online ? "在线" : "离线")}");
    
    // 触发状态变化事件
    try
    {
OnlineStateChanged?.Invoke(IsOnline);
    }
    catch (Exception ex)
    {
     LogHelper.Error(ex, "触发在线状态变化事件异常");
    }
}
```

#### TCP/服务器消息处理增强
- 在线时自动拦截所有通讯触发
- 记录日志："系统离线中，收到TCP消息但不执行操作"
- 不触发相机，不执行检测流程

### 2. 主界面 - UI 控制和状态显示

#### 关键功能
? **系统在线时**: `toolMain.Enabled = false` (整个工具栏禁用)
? **系统离线时**: `toolMain.Enabled = true` (工具栏可用)
? **状态栏显示**: 实时显示系统状态（绿色在线/红色离线）
? **菜单文本**: 动态显示相反状态（点击后切换）
