## ?? 硬件要求对照

### GPU (GC)
```
? 任何支持 CUDA 的 NVIDIA GPU
? 或使用 CPU 模式（性能较低）
? 无需额外安装
```

### OpenVINO
```
? Intel Core 第 6 代及以上
? Intel 集成显卡
? Intel Movidius VPU
?? 需要安装 OpenVINO Toolkit
```

### TensorRT
```
? NVIDIA GPU (GTX 10 系列及以上)
? CUDA 11.x / 12.x
? cuDNN 8.x
?? 需要安装 TensorRT 8.x
```

## ?? 常见错误快速修复

### "没有检测到可用设备"

```
错误可能原因：
├─ GPU 驱动未安装
│  └─ 解决：安装最新 NVIDIA 驱动
├─ CUDA 未配置
│  └─ 解决：安装 CUDA Toolkit
├─ OpenVINO 未安装
│  └─ 解决：下载并安装 OpenVINO
└─ TensorRT 未配置
   └─ 解决：安装 TensorRT 并配置环境变量
```

### 快速诊断命令

```batch
# Windows 检查 GPU
nvidia-smi

# 检查 CUDA
nvcc --version

# 检查 OpenVINO
echo %INTEL_OPENVINO_DIR%

# 检查 TensorRT
where tensorrt
```
